# spring-cloud-demo

#### 介绍
跟着视频学习的全部示例,包含springcloud 生态的各种解决方案(包含流行,停更,以后推荐等)的示例代码

同时包含了自己的一些思考与一些较好博客的知识整合，如：

- 规划抽取公共类-通用接口
- 怎么用aop解决服务降级带来的兜底方法耦合与实现问题
- 怎么使用hystrix dashboard 监控集群问题
